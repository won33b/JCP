<p>도널드 트럼프 전 대통령은 재임 시절 우군 공화당 내의 정적들을 가리켜 종종 &lsquo;리노(Rino)&rsquo;라고 조롱하곤 했다. Rino는 &lsquo;Republican in name only&rsquo;의 줄임말로 &lsquo;이름만 공화당원&rsquo;이란 의미다.</p><!--MA-1-NEW//-->

<p>민주당원인 조 바이든 대통령 역시 &lsquo;디노(Dino&middot;Democrats in name only)&rsquo;, 즉 &lsquo;이름뿐인 민주당원&rsquo;이라고 욕설을 퍼붓고 싶은 사람들이 있을 것으로 보인다. 지난 1월 취임 벽두부터 자신의 국정과제에 사사건건 발목을 잡은 민주당 소속 상원의원 두 명이 있다.</p>

<p>문제의 인물은 웨스트버지니아주 출신인 조 맨신 상원의원(74)과 애리조나주 출신 키어스틴 시네마 상원의원(45)이다. 두 의원의 강력한 반대로 바이든 대통령이 도로와 철도, 공항, 항만 건설, 광대역통신망 확충 등을 위해 제안한 사회간접시설 확충안의 2조 달러 예산이 약 1조 달러로 쪼그라들었다. 향후 10년에 걸쳐 기후위기 대응을 위한 사회안전망 예산과 노인 의료보험 확대, 2년제 대학 및 유치원 무상교육 등에 대한 3조5000억 달러 규모의 사회복지 예산안도 대폭 삭감이 불가피하다. 막대한 재원 조달을 위해 바이든 대통령은 기업 법인세 인상과 고소득자에 대한 증세를 중점 과제로 내걸었다. 그러나 시네마 의원이 증세에 반대하면서 차질을 빚게 되었다.</p>

<p>고작 두 명의 민주당 상원의원 때문에 바이든 대통령의 핵심 국정과제가 진행되지 못하는 상황이 얼핏 이해되지 않을 수도 있다. 하지만 현재 미국 의회 내 공화&middot;민주 양당의 의석 구조를 보면 전혀 이상할 게 없다. 미국은 상하원 양원 구조로 현재 하원은 민주당 220석, 공화당 212석으로 민주당이 다수당이다. 문제는 상원이다. 100석인 상원을 공화&middot;민주 양당이 각각 50석씩 양분하고 있다. 공화당이 상원에서 합법적 의사진행방해 절차인 필리버스터를 펼치면 법안 통과가 어렵다. 민주당이 필리버스터를 무력화하려면 공화당 의원 10명의 동조를 얻어내야 한다. 그러나 &lsquo;바이든 예산안&rsquo;을 결사반대하는 공화당 상원의원 가운데 10명이 떨어져 나올 가능성은 없다. 다른 방법이 있기는 하다. 예산 부문의 법안은 상원의 단순 과반으로 처리 가능하다(예산조정 절차). 그러나 이 절차는 민주당 상원의원 50명 가운데 단 한 사람이라도 반대하면 성사되지 않는다.</p>

<p>이런 상황에서 맨신과 시네마 두 민주당 상원의원이 바이든의 야심찬 양대 예산안에 반기를 든 것이다. 바이든은 울며 겨자 먹기로 두 의원의 주장에 따라 예산액을 대폭 삭감할 수밖에 없었다. 두 의원 중 맨신은 합리적인 편이다. 예산안의 문제점을 구체적으로 지적하고 자신의 뜻이 관철되면 합의해주는 식이다. 맨신은 바이든의 제안인 &lsquo;법인세 28% 인상&rsquo;에 반대하는 대신 &lsquo;25% 인상&rsquo;을 대안으로 제시하기도 했다.</p>

<p>하지만 시네마 의원은 증세에 반대한다는 원칙만 내세울 뿐 구체적 대안을 꺼내지도 속내를 밝히지도 않는다. 시네마 의원은 여러모로 튀는 의원이다. 바이든 대통령의 국정과제나 당론에 거스르는 쪽에 자유롭게 표를 던진다. 의사당에 종종 특이한 옷차림으로 등장해 화제로 떠오르거나 기자들의 질문을 무시해 반발을 산 적도 많다.</p>

<p>변호사 출신인 시네마는 공화당 아성으로 통하는 애리조나주에서 여성으로선 최초로 연방 상원의원에 선출된 인물이다. 양성애자를 자처하는 시네마 의원은 애리조나주 의회 의원으로 활동하던 시절, 성소수자들의 권리를 적극 옹호하면서 유명해졌다. 하지만 시네마의 정치 성향은 민주당 내에서 가장 보수적인 편으로 분류된다. 실제 그의 의회 투표 기록을 보면 공화당 내 대표적 온건파인 리사 머카우스키 의원, 수전 콜린스 의원보다 더 보수 성향을 보인다. 전임 트럼프 행정부 시절 시네마는 트럼프의 국정 철학이 반영된 공화당 법안 가운데 절반 이상에 찬성표를 던졌다. 애리조나주의 민주당 인사들은 2024년 의회 중간선거 때 그를 낙선시켜야 한다고 주장한다.</p>

<p>그러나 시네마 의원은 꿈쩍도 않는다. 여론조사 전문기관 &lsquo;모닝콜&rsquo;의 애리조나 주민들에 대한 현지 여론조사를 보면 그의 지지율은 한때 크게 떨어졌으나 최근엔 50%를 넘기는 등 비교적 높은 편이다. 2024년 민주당 예비경선에서 그를 낙마시키기가 쉽지 않을 것이라는 관측이 지배적이다. 게다가 미국 정계에서 현직 상원의원을 밀어내기란 상당히 어렵다.</p>

<h3><strong>두 의원 묶어 &lsquo;맨셰마&rsquo;라는 신조어 나오기도</strong></h3>

<p>일각에선 시네마 의원이 이처럼 버티는 데는 믿는 구석이 있는 게 아니냐는 의혹도 있다. 시사교양지 〈애틀랜틱〉은 최근 호에서 &ldquo;민주당 간판으로 상원의원 재선을 바라는 게 그의 속셈이 아닐 수도 있다. 그가 부유한 기부자들의 네트워크를 구축해왔다는 사실에 주목할 필요가 있다&rdquo;라고 지적했다. 즉 그는 언제든 이들의 든든한 정치자금을 발판 삼아 민주당을 버리고 무소속으로 전향한 뒤 내년 11월 의회 중간선거에서 공화당이 상원 다수당이 되면 언제든 공화당 측과 연대할 가능성이 크다는 것이다. 뉴욕의 진보적 웹 전문 시사지 〈인텔리겐서〉는 맨신과 시네마 두 의원의 합성어를 뜻하는 &lsquo;맨셰마(Manchema)&rsquo;라는 신조어를 만든 뒤, &ldquo;바이든 대통령의 국정 우선순위가 이들의 변덕에 볼모로 잡혀 있다&rdquo;라고 지적했다.</p>

<p>&lsqlsquo;맨셰마&rsquo;의 활약(?)으로 인해 바이든 대통령은 당초 3조5000억 달러 규모의 사회복지 확충 예산안을 약 1조7500억 달러로 대폭 축소할 수밖에 없었다. 이에 따라 2년제 대학 무상교육안 등 상당수 사회복지 프로그램이 삭제되었다.</p>

$(window).scroll(function() {
	if( $('#wingBanner .wing-banner').length > 0 ){											// 날개배너가 있으면 작동

		var scrollingHeight_L = nonScrollingHeight_L = scrollingHeight_R = nonScrollingHeight_R = 0;

		$('#wingBanner .wing-banner').each(function(idx){
			var $_this		= $(this),														// 날개배너
				scroll		= $_this.data('scroll'),										// 날개배너 스크롤 여부
				topMargin	= $_this.data('top-margin'),									// 날개배너 상단여백
				downMargin	= $_this.data('down-margin'),									// 날개배너 하단여백
				duration	= $_this.data('duration'),										// 날개배너 스크롤 속도
				direction	= $_this.data('direction'),										// 날개배너 위치
				// bannerTop	= $_this.offset().top,											// 날개배너 상단 위치 (가변)
				bannderHeight = $_this.outerHeight();										// 날개배너 높이 (고정)

			var fixWingBannertop	= $('#wingBanner').offset().top;						// 날개배너 상단 위치 (고정)

			var articleTitleHeight	= $('#article-header-title.scroll').height();			// 날개배너 스크롤 시 상단 타이틀 높이 (기사 본문 시)
			if(!articleTitleHeight) articleTitleHeight = 0;

			var $_userFooter		= $('#user-footer'),									// 사이트 Footer
				userFooterHeight	= $_userFooter.outerHeight();							// 사이트 Footer 높이 (고정)

			var winScrollTop		= $(window).scrollTop(),								// 윈도우 스크롤 위치 (가변)
				winHeight			= $(window).height(),									// 윈도우 높이
				docHeight			= $(document).height();									// 페이지 높이

			switch (direction) {															// 날개 배너 위치 별
				case 'left':
					if(scroll == 'y') scrollingHeight_L += bannderHeight;					// 순서대로 노출되는 날개배너 중 스크롤 하는 배너의 높이값 (좌)
					else nonScrollingHeight_L += bannderHeight;								// 순서대로 노출되는 날개배너 중 스크롤 하지 않는 배너의 높이값 (좌)
					break;

				case 'right':
					if(scroll == 'y') scrollingHeight_R += bannderHeight;					// 순서대로 노출되는 날개배너 중 스크롤 하는 배너의 높이값 (우)
					else nonScrollingHeight_R += bannderHeight;								// 순서대로 노출되는 날개배너 중 스크롤 하지 않는 배너의 높이값 (우)
					break;
			}

			if(scroll == 'y') {																// 스크롤 배너만 움직임 처리
				var bannerAllHeight = siteAllHeight = 0;

				// 좌우측 각각 배너 스크롤 처리를 위해 나눠서 처리
				if(direction == 'left'){																			// 좌측 날개
					var wingBnTop_L = winScrollTop - fixWingBannertop + topMargin - nonScrollingHeight_L;			// 날개배너 스크롤 위치 (가변)

					if( articleTitleHeight ) wingBnTop_L += articleTitleHeight;										// 기사본문 시 상단 스크롤 타이틀이 나타나면 스크롤 타이틀 만큼 날개배너 위치 조정

					if((fixWingBannertop + nonScrollingHeight_L) < winScrollTop) {

						bannerAllHeight = articleTitleHeight + scrollingHeight_L + 1;								// 상단 타이틀 + 스크롤배너의 총 높이
						siteAllHeight = winHeight - userFooterHeight + downMargin;									// 윈도우 높이 + 사이트 Footer 높이

						if( (winScrollTop >= ((docHeight - userFooterHeight) - winHeight)) && (bannerAllHeight >= siteAllHeight) ) {					// 사이트 Footer 넘어가는 경우 처리
							wingBnTop_L -= ((userFooterHeight + downMargin) - (winHeight - (scrollingHeight_L + topMargin + articleTitleHeight)));
							$('#wingBanner .wing-banner-left .wing-banner.scroll-on').animate({top:wingBnTop_L+"px" },{queue: false, duration: duration});		// 마지막 배너 움직임 처리
						}
					} else {
						wingBnTop_L = 0;																			// 최상단 처리
					}
					$_this.animate({top:wingBnTop_L+"px" },{queue: false, duration: duration});

				} else {																							// 우측 날개

					var wingBnTop_R = winScrollTop - fixWingBannertop + topMargin - nonScrollingHeight_R;			// 날개배너 스크롤 위치 (가변)

					if( articleTitleHeight ) wingBnTop_R += articleTitleHeight;										// 기사본문 시 상단 스크롤 타이틀이 나타나면 스크롤 타이틀 만큼 날개배너 위치 조정

					if((fixWingBannertop + nonScrollingHeight_R) < winScrollTop) {

						bannerAllHeight = articleTitleHeight + scrollingHeight_R + 1;								// 상단 타이틀 + 스크롤배너의 총 높이
						siteAllHeight = winHeight - userFooterHeight + downMargin;									// 윈도우 높이 + 사이트 Footer 높이

						if( (winScrollTop >= ((docHeight - userFooterHeight) - winHeight)) && (bannerAllHeight >= siteAllHeight) ) {					// 사이트 Footer 넘어가는 경우 처리
							wingBnTop_R -= ((userFooterHeight + downMargin) - (winHeight - (scrollingHeight_R + topMargin + articleTitleHeight)));
							$('#wingBanner .wing-banner-right .wing-banner.scroll-on').animate({top:wingBnTop_R+"px" },{queue: false, duration: duration});		// 마지막 배너 움직임 처리
						}

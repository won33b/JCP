    function getCookie(cookieName) {
        var search = cookieName + "=";
        var cookie = document.cookie;

        if (cookie.length > 0) {
            startIndex = cookie.indexOf(cookieName);
            if (startIndex != -1) {
                startIndex += cookieName.length;
                endIndex = cookie.indexOf(";", startIndex);
                if (endIndex == -1) endIndex = cookie.length;
                return unescape(cookie.substring(startIndex + 1, endIndex));
            } else {
                return false;
            }
        } else {
            return false;
        }
    }

    function setCookie(cookieName, cookieValue, expireDate) {

    }

        $('#prevbtn').show();
        $('#nextbtn').show();
        $('#cardNewsImg' + targetHotNo).show();
        if (targetHotNo == 1) $('#prevbtn').hide();
        if (targetHotNo == totCnt) $('#nextbtn').hide();

11월1일부터 4주 동안 단계적 일상회복(위드 코로나) 1단계로 방역체계가 전환된다. 접종 여부와 관계없이 수도권 10명, 비수도권 12명까지 모일 수 있고&nbsp;식당·카페 등 대부분의 다중이용시설&nbsp;운영시간 제한이 풀려 24시간 영업이 가능하다. 다만 식당·카페에서 예방접종을 마치지 않은 미접종자는 4명으로 제한된다.<br><br>일상회복과 더불어 안전한 시설이용과 미접종자를 보호하기 위해 헬스장·노래방 등 실내체육·유흥시설과 치료 목적 외 의료기관·요양병원을 출입할 때는 접종증명·음성확인제(방역패스) 즉 '백신패스'가 시행된다<br><br>.29일 중앙재난안전대책본부(중대본)는 이 같은 단계적 일상회복 이행계획을 발표했다. 정부는 11월부터 6주 동안 총 3차례에 걸쳐 단계적으로 방역 조치를 완화해간다는 계획이다.<div></div><div><br>다음은 단계적 일상회복과 관련한 질의응답을 정리한 것이다.</div><div><br></div><b>-수도권 식당·카페에서도 접종 여부 관계없이 10명 모일 수 있나.</b><br><div>그동안 사회적 거리두기 기준상 미접종자는 최대 4명을 전제로 수도권은 최대 8명, 비수도권은 최대 10명의 모임이 가능했다. 앞으로 대부분의 다중이용시설에서 접종 여부와 관계없이 수도권 10명, 비수도권 12명의 모임이 가능하다.<br><br>위드 코로나로 인해 사적모임 인원 제한이 대부분 완화됐지만&nbsp;식당과 카페는 마스크를&nbsp;벗고 대화와 음주·식사를 하므로 미접종자 이용 제한기준이 남았다.<br><br>식당·카페는 그대로 미접종자가 4명을 넘을 수 없다. 수도권에서 10명 모인다면 접종자는 6명 이상이어야 하고 비수도권의 12명 중에 접종자가 8명은 돼야 한다. 3차 개편 때 식당·카페 이용과 사적모임 인원 제한 규제 등이 해제될 방침이다.<br><br><b>-1차 개편 때 영업시간 제한이 풀리는 시설은.<br></b>유흥시설을 제외한 생업시설 대다수다. 식당·카페와 학원, 독서실·스터디카페, 영화관·공연장, 실내체육시설, 결혼식장·장례식장, 야구장 등 경기장, 마트·백화점, 노래연습장, 목욕장업, PC방 등이다.</div><div><br>유흥시설, 클럽, 콜라텍, 무도장 등은 1차 개편에 따라 자정까지 운영할 수 있다. 다만 1차 개편 시 접종증명·음성확인제(방역패스)를 도입했고, 영업시간 제한은 2차 개편 때 해제된다.</div><div><br></div><div><b>-접종완료자가 헬스장을 가려면 어떤 절차가 필요한가.</b><b></b></div><div>예방접종완료자는 COOV 앱 또는 전자출입명부의 증명서, 예방접종증명서(종이), 신분증에 부착된 접종 스티커 중 하나를 신분증과 함께 출입시 보여주면 된다.</div><br>유전자증폭(PCR) 음성확인자는 문자 통지서, 보건소에서 발급받은 코로나19 PCR 음성확인서, COOV 앱, 또는 전자증명서 등을 신분증과 함께 보여주면 된다.<br><br>만 18세 이하 청소년은 신분증만 보여주면 된다. 코로나19 확진 후 격리해제자, 의학적 사유로 접종이 어려운 사람 등은 예외자다. 보건소에서 발급한 예외 증명서와 신분증을 제시하면 된다. 단&nbsp;시설별로 증명수단이 다를 수 있고 수단별로 효력 기간이 다르다.<br><br><div><b>-접종완료자만 갈 수 있는 장소도 있나.</b><br><div>접종완료자와 완치자에 한해 출입하는 곳이 있다. 완치자 대부분은 치료받고 90일 이후 권고에 따라 예방접종을 하고 있어 접종완료자기이도 하다. 유흥시설은 접종자만 출입 가능해&nbsp;예외 인정자라도 출입하지 못한다.<br><br>경마·경륜·경정/카지노 그리고 감염취약시설 중 의료기관, 요양병원·시설은 입원 환자나 시설 입소자를 면회할 때 접종완료자와 음성 확인자만 출입할 수 있다.<br></div></div><div><br></div><div><b>-방역패스는 11월1일부터 바로 적용되나.</b><b></b></div><div>제도 초기&nbsp;현장의 적응을 위해 11월1~7일 1주일 동안 계도기간을 운영하고 방역패스 활용에 대한 홍보를 강화할 예정이다. 실내체육시설은 미접종자의 이용권 환불 문제나 현장 혼란 최소화를 위해 11월1∼14일 2주일 동안 계도기간을 운영한다.</div><br>시설 관리자는 접종증명 또는 PCR 음성확인을 위해서는 되도록 상대적으로 위·변조가 어렵고 역학조사 동선추적 편의성이 높은 QR코드를 유도해야 한다. 불가피한 경우 육안으로 확인할 수 있다.<div><br></div><div><b>-영화관에서 팝콘, 야구장에서 치맥을 먹을 수 있나.</b></div><div>영화관이나 야구장 등 스포츠 경기 관람 중의 음식 섭취는 접종완료자만 모이는 경우로 1차 개편 때 시범 허용해보고 위험성을 평가한다. 우선 영화관, PC방, 야구장은 미접종자도 이용할 수 있지만 접종 완료자와 PCR 음성 확인자만 이용할 경우 좌석 띄우기, 인원 제한은 해제한다.</div><div><br>실내외 스포츠 경기는 1차 개편으로 접종 여부와 상관없이 정원의 50%까지 관람할 수 있다. 접종완료자 전용 구역은 100% 정원을 채울 수 있고 취식도 가능하지만 접종 여부와 상관없이 큰 목소리를 내며 응원할 수는 없다.</div><div><br></div><b>-결혼식 하객은 얼마나 초청할 수 있고, 예배에 몇 명까지 모일 수 있나.</b><div><div>결혼식의 경우, 1차 개편 때 미접종자 49명을 포함한 최대 250명 부를 수 있다. 방역패스를 확인했다면 500명 미만으로 치를 수 있다. 2차 개편 때&nbsp;접종완료자만 온다면 인원 제한은 따로 없다. 예배 등 종교 활동은 접종 구분없이 수용 인원의 50%까지 진행할 수 있고 접종완료자만 모이면 인원 제한은&nbsp;없다.</div></div><div><br></div><b>-재택치료는 어떤 처치를 받게 되나. 또 근로자는 생활비 지원을 받을 수 있나.</b><br>재택치료 대상자는 1일 2회 체온, 산소포화도 등 건강관리 모니터링을 받게 된다. 대상자는 모바일앱에 매일 건강정보를 입력하고 1일 1회 이상 의료진과 유선으로 통화하게 된다. 체온계, 산소포화도측정기는 재택치료 키트로 제공된다. 증상이 변화할 경우 의료진 판단 하에 모니터링 횟수를 추가할 수 있다.<br><br>입원·시설치료자와 동일하게 유급휴가비용 또는 생활지원비 지원 대상이다. 공동격리자에 대한 생활지원비는 별도 지원하지 않는다. 지원비는 가구원 수에 따라 다르다. 1인(47만4600원), 2인(80만2000원), 3인(103만5000원), 4인(126만6900원), 5인 이상(149만6700원)이 기준이다.<div><br></div><b>-2차, 3차 개편은 언제 예정인가.</b><br><div>2차 개편은 대규모 행사를 허용할&nbsp;지 결정하거나&nbsp;실외 마스크 착용 지침 해제를 검토할 계획이다. 시점은 12월 중순으로 그 때면 국민 접종 완료율이 80% 이상 올랐을 것이다. 현재도 실외에서 다른 사람과 2m 이상 거리를 두면 벗을 수 있다. 3단계는 사적모임 인원 제한 해제를 검토하는데&nbsp;내년 1월쯤은 되어야 할 전망이다. 15명 안팎의 단체 회식은 그 이후 가능할 것으로
